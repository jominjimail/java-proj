필수 기능

- 피자에 토핑을 추가할 수 있다. 
- 토핑이 추가된 피자의 가격과 조리시간, 메뉴정보를 알 수 있다.
- 피자가 오븐에 들어가서 조리가 완료되면 알림이 울린다.
- 손님이 원하는 메뉴를 주문받는다.
